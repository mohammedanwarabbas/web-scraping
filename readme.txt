This is a Python Django web application. This application will help in performing web scraping. Result can be downloaded optionally.
To run this project one should have a python and django installed in their computer. Also all the requirements from requirements.txt file need to be installed inorder to ensure proper running of the application. 
command to run django project : "python manage.py runserver". 
command to install all the packages from requirements.txt : "pip install -r requirements.txt".
IMPORTANT NOTE : Web scraping of some one else's website without their concent is not legal and if found will be punished.
